# Цель проекта 
Создание эффективного матричного калькулятора с удобным пользовательским интерфейсом.

# Описание проекта
Основная функция приложения - проведение математических операций над матрицами.

# Сроки
1 модуль:
  - создание интерфейса приложение
  - добавление нескольких простейших операций над матрицами 
2 модуль: 
  - добавление остальных операций над матрицами в функционал прилоджения 
  - доработка существующих алгоритмов
# Стек технологий
C++
MS VisualStudio
